# Telegram Mini App

Статический Telegram Mini App для прогнозов на матчи.

## Запуск локально

```powershell
python -m http.server 5173 --bind 127.0.0.1
```

Открыть:

```text
http://127.0.0.1:5173/
```

## Что работает

- список прогнозов с фильтрами;
- экран детального прогноза;
- статистика;
- профиль с данными из `Telegram.WebApp`, если приложение открыто внутри Telegram;
- админ-панель для добавления прогнозов;
- сохранение demo-данных в `localStorage`.

Для публикации в Telegram нужен HTTPS URL. Подойдет любой статический хостинг или туннель для разработки.
